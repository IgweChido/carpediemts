# carpediemts

Frontend developer test implementation built with React, TypeScript, Vite, and Tailwind CSS.

## Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS

## What is included

- Responsive shop dashboard layout based on the provided design
- Sidebar, top navigation, job table, job details panel, and scheduling modal
- Basic UI states for loading, empty, and error scenarios
- Typed mock data and a small state hook to simulate dashboard loading

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
